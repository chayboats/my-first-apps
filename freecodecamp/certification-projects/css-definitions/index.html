A

  • Alpha Channel : (rgba) Similar to opacity.

    • Ex: rgba(255, 255, 255, 50%)
B

  • Block Elements : Elements that stack when next to one another instead of being side by side.

    • Div is a block element. (You can use display: inline-block to cancel this).

  • Box Shadow : A property that places a shadow behind an element.

    • Box-shadow: offsetX offsetY blurRadius spreadRadius color;
    • Box-shadow: 5px 5px 5px 5px red;
    • BlurRadius is the value inside of the box shadow blurring the radius
    • SpreadRadius spreads the radius out farther
    • #px #px color
C

  • Class Selector : Located in the CSS files. Like a type selector, but it has a dot before the name. YOU MUST add this class to the HTML FILE!

    • Ex in HTML: <div class=”menu”>
    • Ex in CSS:
      .menu {
      background-color: blue;
      }

  • Color Stop : Adding a % to a color in the gradient makes it appear in the transition longer.

    • Ex: (180deg, rgb(255, 0, 0) 0%, rgb(0, 255, 0,) 50% rgb(0, 0, 255) 100%);

    • Purple is the automatic gradient direction and color stop, so these values do not need to be inputted.

  • Complementary Colors : Two colors that are opposite from each other on the color wheel.

    • Two complementary colors mixed together create a gray.

  • CSS : Cascading Style Sheets, a language used to style HTML elements.

  • CSS Linear-Gradient : A function that lets you control the direction of the transition along a line, and which colors are used.

    • linear-gradient(gradientDirection, color1, color2, ...);
    • Creates an image element
    • Is usually paired with background property that accepts and image as a value
    • To represent the colors, you can use any type of color key using hex, rgb, or hsl.
    • Ex: .red {
      background: linear-gradient(90deg, rgb(255, 0, 0), rgb(0, 255, 0));
      }

  • CSS Selector : The part of a CSS rule-set that defines which HTML elements will get the design.

E

  • Element Selector : Selecting all elements from the body to style in CSS.

F

  • Fallback Font Value : Adding another font with a comma after a font family (in case the first is not available)

    • Ex: In font-family: arial, georgia; (Georgia is the fallback font value)

  • Flexbox : A display value that creates a container to position elements within the container.

    • See Notes about Flexbox

  • Function : A piece of code that can take an input and perform a specific action.

G

  • Global Values : Values that can be used for many elements and classes in CSS.

    • Ex: initial, inherit, unset, revert, revert-layer

  • Gradient : Color Transition

  • Gradient Direction : Degree that the gradient is going.

    • Ex: 180deg
H

  • Hex Values (Hexadecimal) : Start with a # character and take six characters from 0-9 and A-F. The first pair of characters represent red, the second pair represent green, and the third pair represent blue.

    • #4B5320

    • Helpful Links: Find Hex from Image ; Color Contrast Ratio

  • HSL (Hue, Saturation, and Lightness) : A function that accepts 3 values: a number from 0 to 360 for hue, a percentage from 0 to 100 for saturation, and a percentage from 0 to 100 for lightness.

    • 0 is red; 120 green; 240 blue
    • 0-100% saturation (gray-pure color)
    • 0-100% lightness (black, neutral, white)
    • Ex: hsl(240, 100%, 50%)

  • Hues : Colors

I

  • Inline Elements : The opposite of block elements. Elements that are next to one instead of being stacked like blocks.

    • Labels are inline elements. (You can use display: block to cancel this.)
M

  • Media Query : @media - A rule that allows you to make circumstantial styling changes.

    • Ex: @media (max-width: 800px) {
      #gallery img {
      width: 50%;
      }
      }
    • When the width is less than 800px, put the image to 50%
    • You can also add and to use to rules
    • Ex: (min-width 2000px) and (max-width: 800px)
    • Sometimes the Media Query will need to be at the end of the css page.
O

  • Opacity : How opaque or transparent something is.(0-1.0 or 0%-100%)

P

  • Primary Colors : Red, Green, or Blue set to 255.

  • Properties : Parts of an element or class that you can choose to style.

  • Pseudo Element : Used to applying something to a certain part of an element.

    • Typically begins with to colons after the element
    • Ex: I want the line of the paragraph to be blue
    • p::first-line: {
      color: blue;
      }
R

  • RGB : A coloring value with 3 augments--(red, green, and blue). The values for each go up to 255.

    • 0 for all is black
    • Ex: background-color: rgb(255, 0, 255) [this should be a purple (red and blue)
S

  • Secondary Colors : 2 Primary Colors set to 255.

T

  • Tertiary Colors : 1 max (255) 1 half (127)

  • Type Selector : Selecting an element from the body to style in CSS.

    • Ex: h1 {
      Background-color: blue;
      }
